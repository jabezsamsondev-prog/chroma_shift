# Chroma Shift

**Chroma Shift** is a cyberpunk-themed reflex training game designed to test your precision, speed, and control. Built with pure Vanilla JavaScript, it offers a high-performance, responsive experience without the overhead of frameworks.

## üéÆ Gameplay

The objective is simple but challenging: **Tap the numbers in ascending order.**

- **Precision:** Hit the correct number to build your combo.
- **Speed:** Race against the clock to complete the sequence.
- **Control:** Avoid penalties for incorrect taps.

## üèÜ Difficulty Levels

- **STANDARD (5x5):** 30s limit. The baseline training module.
- **PRO (5x5):** 22s limit. Faster pace for seasoned agents.
- **ELITE (5x5):** 18s limit. High-speed reflex testing.
- **OMEGA (6x6):** 40s limit. The ultimate challenge. Features the **Shift** mechanic: the grid reshuffles every 6 correct taps.

## ‚ú® Features

- **No Frameworks:** Pure HTML, CSS, and Vanilla JavaScript.
- **Dynamic Visuals:** Particle effects, screen shake, and smooth animations.
- **Audio Feedback:** Synthesized sound effects using the Web Audio API.
- **Progress Tracking:** LocalStorage saves your best times for each difficulty level.
- **Responsive Design:** Optimized for both desktop and mobile devices.
